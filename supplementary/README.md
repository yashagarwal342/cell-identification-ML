# Supplementary Folder Description

This folder contains files and code generated during the project development process that were not used in the final solution.

It includes the following subfolders:

- **old_EDA**: Stores old exploratory data analysis (EDA) files, including scripts and notes for data cleaning, preprocessing, and initial analysis.
- **old_model_code**: Stores training and debugging code for old candidate models. The final trained models have been placed in the DELIVERABLES folder.
- **old_shiny**: Stores code for old versions of the Shiny application.
- **projectdata_rest/images**: Stores additional datasets or image data not used in the project.
- **others**: Stores miscellaneous files that do not fall into the above categories.
